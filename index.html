Site en PHP

4. J’ai conçu et développé une application web, appelé Past App, en 17 jours consécutifs. Mais à mon grand étonnement j’ai vu que j’ai perdu le fichier. Mais heureusement j’avais créé un tutoriel le jour de ma dernière ligne de code. Alors le lien du tuto est là. J’ai voulu créé un mini plateforme pour les pasteurs en Haïti. L’explication du tutoriel est en créole Haïtien, mais vous pouvez observer les manœuvres que je fais. Cette application Web a 3 grandes parties :
a) Une partie grand public, ou les visiteurs peuvent rechercher un pasteur selon plusieurs critères (son nom, le nom de son district, le nom de la zone)
b) Une partie pour les utilisateurs (les pasteurs). Ou les utilisateurs inscrits peuvent changer leurs informations personnelles, poster sur leurs murs et faire ce les visiteurs peuvent faire.
c) Une partie pour les administrateurs, avec plusieurs niveaux de privilège selon la position de l’administrateur dans l’organisation.
